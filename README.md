O SpotTI é um reprodutor de musica local com referência ao Spotify (Spot) e a área de TI que estudamos (TI).
Seu objetivo é poder tocar musicas facilmente baixadas em seu dispositivo sem anúncio, com 5 músicas já pré baixadas para ouvir.


# Funcionamento da aplicação

1°  Começe abrindo o arquivo cadastro.py, é o início de nossa aplicação. É a partir de lá que serás redirecionado para o resto das abas. Lá você terá que fazer um cadastro se não tem conta ainda, ou o login se já tiver.



2°  A proxima aba é o menu principal, no qual as musicas pré baixadas se encontram e o butão para salvar suas musicas locais. Ou seja, so clicar nas musicas que estão prontas para serem ouvidas.


3°  Há uma imagem do lado superior esquerdo da tela. Tal imagem redirecionará você para a tela do seu logim, com seu nome e sua senha.
